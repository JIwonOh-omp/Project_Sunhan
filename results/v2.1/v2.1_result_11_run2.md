### (3) **피부타입 주의**  
- **민감성 피부**:  
  - Contains fragrance. Patch test recommended for sensitive skin.  
  - Contains **Salicylic Acid** and **Caprooyl Salicylic Acid** – mild exfoliants that may irritate sensitive skin.  
  - Contains **EU-listed fragrance allergens**: **Limonene**, **Citronellol**, **Geraniol**, **Citral**, **Linalool**. Patch test recommended.  
- **건성 피부**:  
  - Contains **Salicylic Acid** (BHA), which can be drying.  
- **지성/여드름성 피부**:  
  - Contains **Salicylic Acid** – helps control excess sebum and exfoliate pores.  
  - Contains **Charcoal Powder** – may help absorb excess oil.
