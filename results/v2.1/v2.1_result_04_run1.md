### **3. Skin Type Alerts**  
- **Sensitive Skin**  
  - Contains **essential oils** (e.g., **Litsea Cubeba Fruit Oil**, **Lavandula Angustifolia Flower Extract**), which may irritate sensitive skin.  
  - Contains **limonene** and **citral** (EU-listed fragrance allergens). Patch test recommended.  
- **Dry Skin**  
  - Contains **Denatured Alcohol** (not listed, but **2,3-Butanediol** is a low-risk humectant).
