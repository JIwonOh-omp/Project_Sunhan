# §성분명 대응표 (Ingredient Name Map)

출처: 대한화장품협회 성분사전 표준화명칭목록(2026.06.30 기준)으로 대조 검증 완료.
용도: 오역 위험이 높은 한국어 성분명 → 영어 INCI 고정 매핑.

## 매칭 규칙
- 라벨의 한국어 성분명이 아래 표에 있으면, 반드시 지정된 INCI로 번역한다.
- 표에 **없는** 성분은 지어내지 말고, 확신이 서지 않으면 한국어 원문을 병기하고
  "translation uncertain"으로 표시한다. (자가검증 규칙과 연동)
- "종(species) 불명"으로 표시된 항목은 단일 학명을 단정하지 말고 원문을 병기한다.
